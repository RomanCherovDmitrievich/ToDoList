Подробное описание

Класс для создания и управления анимацией падающих снежинок

Обеспечивает визуальный эффект падающего снега в фоне приложения. Создает множество снежинок с различными параметрами (размер, скорость, прозрачность) и анимирует их движение по экрану.

Заметки
Используется в MainController для создания новогодней атмосферы
Предупреждения
Для корректной работы требуется JavaFX AnimationTimer
См. также
MainController
javafx.animation.AnimationTimer
Начиная с
Версия 1.0

Snowflake view.SnowflakeAnimation.createSnowflake ( )
private

Создает одну снежинку со случайными параметрами

Возвращает
Объект Snowflake с заданными параметрами

Создает графический элемент снежинки (круг) со случайными:

  • Размером (1-3 пикселя)
  • Начальной позицией (вверху экрана)
  • Прозрачностью (30-100%)
  • Цветом (белый, AliceBlue или LightBlue)
  • Скоростью падения (30-100 пикселей/секунду)
  • Параметрами покачивания
Заметки
Снежинка создается выше видимой области экрана для плавного появления
См. также
javafx.scene.shape.Circle
javafx.scene.effect.Glow

◆ updateSnowflakes()

void view.SnowflakeAnimation.updateSnowflakes ( double deltaTime)
private

Обновляет позиции всех снежинок

Аргументы
deltaTimeВремя, прошедшее с предыдущего обновления (в секундах)

Для каждой снежинки:

  1. Двигает её вниз с заданной скоростью
  2. Применяет покачивание из стороны в сторону
  3. Проверяет выход за границы и перераспределяет снежинку
  4. Обновляет прозрачность для эффекта мерцания
Аргументы
deltaTimeДельта времени между кадрами анимации
Заметки
Если снежинка упала за нижнюю границу, она перемещается наверх
Если снежинка вышла за боковые границы, она появляется с противоположной стороны
См. также
createSnowflake()
