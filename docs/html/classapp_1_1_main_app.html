Подробное описание

Главный класс приложения To-Do List.

Класс MainApp является точкой входа JavaFX приложения. Отвечает за инициализацию, настройку и запуск главного окна приложения. Управляет жизненным циклом приложения, загрузкой ресурсов и обработкой ошибок.

Метод инициализации приложения

Вызывается автоматически JavaFX перед методом start() Выполняет предварительную настройку приложения

Заметки
В этом методе:
  • Выводится информация о версиях Java и JavaFX
  • Инициализируется AudioManager
  • Логируются ошибки инициализации
Предупреждения
Исключения в этом методе не прерывают запуск приложения

void app.MainApp.start ( Stage primaryStage)

Метод запуска JavaFX приложения

Основной метод настройки и отображения пользовательского интерфейса

Аргументы
primaryStageГлавное окно приложения, предоставляемое JavaFX
Исключения
IOExceptionесли не удается загрузить FXML файл главного окна
NullPointerExceptionесли не найден CSS файл или иконка
Заметки
В этом методе:
  • Загружается FXML файл главного окна
  • Настраивается сцена с CSS стилями
  • Устанавливается иконка приложения
  • Настраиваются минимальные размеры окна
  • Добавляется обработчик закрытия окна
  • Отображается главное окно
Предупреждения
Все исключения перехватываются и показываются в диалоговом окне
