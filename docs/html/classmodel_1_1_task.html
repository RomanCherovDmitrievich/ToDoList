Подробное описание

Основной класс, представляющий задачу в приложении

Класс Task содержит все данные о задаче: заголовок, описание, даты начала и окончания, приоритет, категорию и статусы выполнения. Предоставляет методы для проверки просроченности, форматирования времени и сериализации в JSON формат для сохранения.

Конструктор для создания новой задачи

Создает задачу с автоматически сгенерированным ID и текущей датой создания

Аргументы
titleЗаголовок задачи (не может быть null или пустым)
descriptionОписание задачи (может быть пустым)
startTimeВремя начала задачи
endTimeВремя окончания (дедлайн)
priorityПриоритет задачи (если null, используется IMPORTANT по умолчанию)
categoryКатегория задачи (если null, используется OTHER по умолчанию)
Исключения
IllegalArgumentExceptionесли title пустой или endTime раньше startTime
Заметки
Приоритет по умолчанию - IMPORTANT, категория - OTHER
ID генерируется с помощью UUID.randomUUID()

Конструктор для загрузки из JSON.

Создает задачу из данных, загруженных из JSON файла Проверяет просроченность при создании

Аргументы
idУникальный идентификатор задачи
titleЗаголовок задачи
descriptionОписание задачи
startTimeВремя начала в строковом формате (yyyy-MM-dd'T'HH:mm:ss)
endTimeВремя окончания в строковом формате (yyyy-MM-dd'T'HH:mm:ss)
priorityПриоритет в строковом формате (enum name)
categoryКатегория в строковом формате (enum name)
completedСтатус выполнения
overdueСтатус просроченности
createdAtДата создания в строковом формате (yyyy-MM-dd'T'HH:mm:ss)
Исключения
java.time.format.DateTimeParseExceptionесли строки дат не соответствуют формату
IllegalArgumentExceptionесли priority или category не являются допустимыми enum значениями
Заметки
Используется JsonUtil для десериализации задач из файла
Автоматически вызывает checkOverdue() после создания
См. также
util.JsonUtil
