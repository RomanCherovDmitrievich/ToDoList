Подробное описание

Менеджер задач - центральный класс для управления задачами

Класс TaskManager реализует паттерн Singleton и служит центральным хранилищем и менеджером для всех задач в приложении. Отвечает за операции CRUD (Create, Read, Update, Delete) с задачами, поиск, фильтрацию и проверку статусов.
