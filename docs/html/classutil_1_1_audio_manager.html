Подробное описание

Менеджер для работы с аудио в приложении

Класс AudioManager реализует паттерн Singleton для управления звуковыми эффектами и фоновой музыкой. Обеспечивает загрузку, воспроизведение и контроль аудио ресурсов приложения. Поддерживает включение/выключение звуков, регулировку громкости и плейлист с автоматическим переключением треков.

Автор
Чернов
Версия

Заметки
Использует JavaFX Media API для воспроизведения звуков
Предупреждения
Звуки требуют наличия соответствующих файлов в resources/audio/
