ToDoList (Cross-platform Edition)

О проекте

Десктопное JavaFX-приложение для управления задачами с SQLite-хранилищем, системными уведомлениями, календарём и фоновой музыкой.

Архитектура

  • app: запуск JavaFX приложения
  • view/viewmodel: UI и представление данных
  • model: сущности и бизнес-логика
  • repository: доступ к SQLite
  • util: JSON, аудио, уведомления, напоминания, темы

База данных

Используется SQLite с 5+ таблицами. Подробности: Схема Базы Данных

Ключевые возможности

  • CRUD задач, фильтрация, сортировка и ручной порядок
  • Редактирование задач на месте и контекстные действия
  • Системные уведомления о старте/просрочке задач
  • Календарь + экспорт в .ics
  • Темы (Summer/Dark/Ocean/Custom)
  • Циклический плейлист из папки /audio
  • Функция "Фокус дня" (топ-3 приоритетных задачи)
  • Кроссплатформенные скрипты setup/run/package
  • Документация по Умный планировщик задач (To-Do List). Последние изменения: . Создано системой 1.15.0
