Утилита для работы с JSON файлами задач

Класс предоставляет статические методы для загрузки, сохранения и управления задачами в формате JSON. Все задачи хранятся в файле tasks.json в директории data/. Класс использует стандартные Java библиотеки для работы с файлами и парсинга JSON без внешних зависимостей.
