Подробное описание

Утилита для загрузки и сохранения задач в формате JSON.

Класс предоставляет статические методы для работы с задачами в формате JSON. Поддерживает гибкую обработку форматов дат, тестирование через изменение путей, и обработку ошибок с резервным копированием.
